# ST-GNN Groundwater Model Implementation Walkthrough

The integration of the Spatio-Temporal Graph Neural Network (ST-GNN) is now complete. The system has been upgraded to support deep-learning-based groundwater estimation, capable of providing advanced predictions with confidence intervals and multi-horizon forecasts.

## 1. ML Pipeline & Feature Engineering
- **Spatial K-NN Graph**: Created `ml/graph_builder.py` to construct a spatial graph connecting villages with nearby piezometers using PyTorch Geometric and NetworkX.
- **Anomaly Detection**: Developed `ml/anomaly_detector.py` to identify and handle outliers in time-series data.
- **Model Architecture**: Built `ml/spatio_temporal_gnn.py` implementing Graph Attention Networks (GAT) for spatial message passing and MLP layers for temporal predictions. We use Monte Carlo dropout for estimating prediction uncertainty.
- **Temporal Lags Extraction**: Upgraded `ml/generate_dataset.py` to automatically compute and export temporal lags (`gw_lag_1`, `gw_lag_2`, `gw_lag_3`, `gw_lag_6`, `gw_lag_12`) relative to the last recorded month. This provides the historical sequence features needed for the temporal aspect of the ST-GNN.

## 2. Backend Integration
- **Inference Service**: Implemented `STGNNInferenceService` in `backend/app/ml/st_gnn_service.py` to handle loading the PyTorch model and executing forward passes. It provides mock inference data for scaffolding if the physical model file is not yet trained.
- **FastAPI Endpoints**: Registered the `/api/predictions/st-gnn/village/{village_id}` endpoint in `backend/app/main.py`. This API returns:
  - Base prediction
  - 95% Confidence Intervals
  - 7, 30, and 90-day short-term forecasts.

## 3. Frontend Visualization
- **Map Uncertainty Visualization**: Integrated the ST-GNN confidence interval spread into the interactive `MapView.jsx`. High-uncertainty areas will now highlight dynamically.
- **Layer Toggle**: Added a "GNN Uncertainty" toggle in the map `Sidebar.jsx` layers control.
- **Insights Panel**: Expanded `VillageInsightsPanel` in `frontend/src/components/UI.jsx` to render the ST-GNN predictions:
  - Dedicated section for the ST-GNN prediction output.
  - Confidence interval bounds explicitly displayed.
  - 7/30/90-day forecast values rendered cleanly alongside the historical context.

## Next Steps
The infrastructure is complete. The next operational step is to train the `st_gnn.pt` model on the newly generated dataset and place it in the `model/` directory for live PyTorch inference.
